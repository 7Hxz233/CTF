技巧提示
  • 本功能在恢复备份数据的同时，将全部覆盖原有数据，请确定恢复前已将论坛关闭，恢复全部完成后可以将论坛重新开放。
  • 数据恢复功能只能恢复由当前版本 导出的数据文件，其他软件导出格式可能无法识别。
  • 从本地恢复数据需要服务器支持文件上传并保证数据尺寸小于允许上传的上限，否则只能使用从服务器恢复。
  • 如果一组备份数据有多个分卷备份文件，请选择该组备份文件的全部分卷进行恢复，用时间前缀来区分不同备份。
